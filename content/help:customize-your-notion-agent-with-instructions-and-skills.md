Instructions and skills are available for workspaces with Notion AI enabled. Learn more about Notion AI and availability [in this article](https://www.notion.com/help/notion-ai-faqs).

Use instructions to set how your Notion Agent shows up every day, and skills to save your favorite prompts so you can run them anytime.

## What are instructions?

Instructions tell your personal Notion Agent how to respond by default (tone, formatting, and rules to follow). Your Agent uses your active instructions page in every chat. You can only use one instructions page at a time, but you can switch it anytime.

* Use instructions for things like tone, writing style, and rules you want followed every time.

* You can only have one active instructions page at a time.

**Examples of instructions**

* “Keep my writing concise and friendly.”

* “Always format action items as a checklist.”

* “When I ask for a summary, keep it under 5 bullets.”

* “End every response with next steps and a checklist.”

* “When I paste meeting notes, summarize decisions first, then action items.”

## What are skills?

Skills are saved prompts you can run whenever you need them. A skill is just a page you’ve marked as a skill, so you can organize and share based on your preferences.

* Use skills for repeatable tasks like “Proofread,” “Explain,” or “Reformat.”

* Skills are pages, so you can organize and share them like any other page in Notion.

**Examples of skills**

* Turn meeting notes into a follow-up email.

* Rewrite a paragraph for a different audience.

* Convert a brainstorm into a project plan.

## Where to manage instructions and skills

You can manage both from settings.

* To manage new instructions and skills, go to `Settings `→ `Notion AI `→ `General`.

* From there, you can view your current instructions, switch to a different instructions page, manage your skills, and discover skills others use in your workspace.

## Create an instructions page

1. Create a new page (or open an existing page).

2. Click `··· `→ Use with AI → `Set as Notion AI instructions`.

3. Write your preferences clearly.

You can also create new instructions in `Settings` → `Notion AI` → `General`.

To remove an instructions page, click`···`.

**Tips for good instructions**

* Start with what you want, not what you don’t want.

* Be specific. If you have a formatting preference, describe it.

* Keep it short. You can always add more later.

## Create a skill

1. Create a new page.

2. Write the prompt you want to reuse.

   * Tip: Write the skill like a reusable request, including the end result you want. For example, “Rewrite this for a customer-facing audience in 3 short paragraphs.”

3. Click `···`→ Use with AI → `Set as skill`.

4. Pin the skill if you want quick access.

You can also create new skills in `Settings` → `Notion AI` → `General`.

To remove a page as a skill, click`···`.

## How to use a skill

You can run skills from a few places, depending on what you’re doing.

* In chat, run a skill by typing @ then the name of the skill page.

* Select a body of text. In the text selection menu, choose a built-in skill like **Improve writing, Proofread**, **Explain**, or **Reformat.**

* You can also designate which skills you want to show up in the text selection menu by going to `Settings`

  → `Notion AI`→ `General`.

## Sharing and permissions

Because instructions and skills are pages, [sharing](https://www.notion.com/help/sharing-and-permissions#share-a-page-with-others) works the same way it does for any other Notion page.

* If you share a skill with someone, they can run that skill once they have access to the skill page. If they can edit the page, their edits will change the skill.

* If you share your instructions page, it does not automatically change anyone else’s Notion Agent. They would need to set it as their own instructions.

* View skills you’ve created, skills shared with you, and skills in your workspace by going to `Settings `→ `Notion AI` → `General`.


## FAQs

### Can I use Notion AI instructions with Custom Agents?

No. <!-- -->Notion Agent instructions only apply to your Notion Agent. [Custom Agents](https://www.notion.com/help/custom-agent) have their own instructions <!-- -->and permissions as part of their setup.


### Will my Notion Agent use skills automatically?

You can run skills directly, and your Agent may also suggest or use relevant skills when it makes sense.
